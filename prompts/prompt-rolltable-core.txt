================================
JSON FIELD RULES
================================

Table fields:
- tableName: non-empty string, maximum 50 characters.
- tableDescription: HTML string.
- tableImagePath: image path or "".
- drawWithReplacement: boolean.
- displayRollFormula: boolean.
- missingDocumentPolicy: "error" or "text". It is used only if a Document name cannot be resolved during import.
- results: non-empty array.

Result fields:
- resultType: exactly "text" or "document" as selected by the profile.
- resultImagePath: image path or "".
- resultDocumentType: canonical lookup type for document results (for example "Actor", "Item", "Spell", "Feature", "JournalEntry", "RollTable", "Scene", "Macro", "Playlist", or "Cards"); use "" for text results.
- resultDocumentSource: "world", an exact compendium pack id from the catalog, or "". It narrows Blacksmith's name-to-UUID lookup; use "" for text results.
- resultText: displayed text. For document results it must exactly match a catalog entry name.
- resultWeight: positive whole number.
- resultRangeLower and resultRangeUpper: positive whole numbers with no overlapping ranges.
