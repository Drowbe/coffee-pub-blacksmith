================================
JSON FIELD RULES & EXPECTATIONS
================================

Replace ALL-CAPS placeholders. All fields are mandatory on every result row.

Table-level:
- TABLENAME: (string) Full table name (max 50 characters)
- TABLEDESCRIPTION: (string) HTML narrative (p, ul, li, b, h2, etc.)
- TABLEIMAGEPATH: (string) set to ""
- DRAWWITHREPLACEMENT: (boolean) true or false (not a string)
- DISPLAYROLLFORMULA: (boolean) true or false (not a string); profiles often set the table default

Per result:
- RESULTTYPE: (string) "text", "document", or "Compendium" (profile fixes this)
- RESULTIMAGEPATH: (string) set to ""
- RESULTDOCUMENTTYPE: (string) Required when resultType is "document" — actor, adventure, card stack, item, journal entry, macro, playlist, rollable table, or scene. Otherwise " " or null per profile
- RESULTCOMPENDIUM: (string) Required when resultType is "Compendium" — compendium pack id (e.g. dnd5e.items). Otherwise " " or null per profile
- RESULTTEXT: (string) Display text. For document rows, must exactly match a world document name. For compendium rows, must exactly match an entry name in resultCompendium
- RESULTWEIGHT: (number)
- RESULTRANGELOWER: (number)
- RESULTRANGEUPPER: (number)
